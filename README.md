# 🌐 遊戲輔助/數位商品電商網站開發藍圖 (提示詞)

**【角色設定】**
你現在是一位資深全端工程師與 UI/UX 設計師，擁有豐富的電商平台與數位產品銷售網站開發經驗。請協助我從零開始打造一個專門販售「遊戲輔助工具/數位商品」的現代化網站。

**【專案技術棧 (建議)】**
*   **前端框架**：Next.js (React) 或 Vite + Vue3
*   **UI/樣式**：Tailwind CSS + Framer Motion (用於流暢的動畫效果)
*   **組件庫**：shadcn/ui 或 Radix UI (打造高質感且可客製化的介面)
*   **資料庫**：PostgreSQL (搭配 Prisma ORM) 
*   **身分驗證**：NextAuth.js (支援 Email/密碼及第三方登入如 Discord/Google)
*   **金流串接**：Stripe (信用卡) 或區塊鏈加密貨幣支付橋接 (視客群需求而定)

**【網站視覺與設計風格 (UI/UX)】**
*   **主色調**：深色模式 (Dark Mode) 為主，營造電競與科技感。
*   **點綴色**：霓虹綠、賽博龐克紫或電競藍 (作為按鈕、標籤與強調色)。
*   **排版**：現代化極簡風格，大字體標題，重點突出產品縮圖與展示影片。

**【核心功能模組】**

1.  **首頁 (Landing Page)**
    *   **Hero Section**：具衝擊力的大圖或自動播放的展示影片，搭配動態 Slogan 及「立即探索」按鈕。
    *   **熱銷/推薦商品**：卡片式展示熱銷輔助工具，包含標籤 (如 "安全未封號"、"自動更新")、價格與購買按鈕。
    *   **平台優勢/特點**：強調安全性、即時交付、24/7 客服支援等特色。
    *   **買家評價 (Testimonials)**：跑馬燈形式展示真實用戶好評。

2.  **商品導覽與搜尋 (Catalog & Search)**
    *   **分類系統**：包含 Fivem輔助、Fivem選單、FiveM解封、Spoofer、Valorant輔助、APEX輔助、CS2輔助、GTA5輔助、三角洲行動、騙子酒館、Roblox輔助、暗區突圍 等具體分類。
    *   **過濾與排序**：價格高低、最新上架、銷量排行。

3.  **商品詳情頁 (Product Detail Page)**
    *   **展示區**：圖片輪播與實機運作影片。
    *   **詳細資訊**：支援的作業系統、版本相容性、功能列表 (如 ESP、Aimbot、巨集等，視具體商品而定)。
    *   **狀態指示燈**：顯示該商品目前的狀態 (如 🟢 正常運作、🟡 更新中、🔴 維護中)。
    *   **購買區塊**：多種授權期限選擇 (天卡、周卡、月卡、永久版) 及價格動態切換。

4.  **會員中心 (User Dashboard)**
    *   **訂單管理**：查看歷史訂單與發票。
    *   **數位資產庫**：購買後自動分發下載連結與啟動金鑰 (License Key)。
    *   **教學與說明**：提供專屬的安裝教學文件或影片。

5.  **後台管理系統 (Admin Panel)**
    *   **商品管理**：新增/編輯商品、設定卡密庫存、更改商品狀態。
    *   **卡密系統 (Key Management)**：批量匯入、生成與狀態追蹤 (未使用、已啟用、已過期)。
    *   **訂單與會員管理**：查看營收報表、處理客戶問題。

**【開發階段拆解】**
*   **第一階段**：初始化專案結構，建立全域 Layout 與深色主題 UI 系統。
*   **第二階段**：開發首頁、商品列表頁與商品詳情頁 (先使用假資料 Mock Data)。
*   **第三階段**：建置資料庫架構與後端 API，串接真實資料。
*   **第四階段**：實作身分驗證與購物車/發卡系統。
*   **第五階段**：串接金流測試與部署上線。
